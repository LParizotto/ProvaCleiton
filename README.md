# Prova Cleiton

## Proposta:
- Criar um aplicativo em React Native.
- Tema livre.
- Ter pelo menos uma funcionalidade.
- Ter um README.

## Aplicativo Criado:
### Gerador de paletas de cores
- O aplicativo criado tem como principal objetivo a geração de uma paleta de cor.
- Para geração das cores, temos um botão no footer "Gerar Nova Paleta".
- Quando clicado, 5 cores aleatórias são mostradas.
- Se o usuário gostar de alguma cor, é possível clicar no cadeado dessa cor, que impedira ela ser sobreposta por outra.
- As cores geradas possuem seu código hexadecimal no meio, podendo ser copiada caso o usuário clique em cima do código.
